# CasualTweaks
Configurable Tweaks that make the game a bit calmer to relax and enjoy.

## Exactly, what is tweaked?
1. Fuel & Energy Capacity are tweaked (2x default increase)
2. Vehicle Max speed are tweaked (4x default increase)
3. Job pricing is tweaked (6x default increase)

All of the above is configurable in some way at `/BepInEx/config/com.laymxd.casualtweaks.cfg`
That's it. thats all this mod does, its really simple.
